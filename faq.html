How to use

Select the desired chart type from the Plugins > Sketch Data Studio menu. Follow on-screen instructions and your chart will be automatically generated.

Data Types

Random 0 - 1

A random decimal number from 0 - 1

Random 0 - 100

A random decimal number from 0 - 100

Set your own random values

Input a comma-seperated list of values that will be picked at random for each row. This can be used to select random numbers (1,2,3,4...) or words (dog, cat, horse...)

Random X

Names randomly chosen from a list of stocks that were found on the internet.

CSV Import

Save a spreadsheet in .csv format and import it through the dialogue. Only files with a .csv or .CSV extension will be importable. (Note - data must be in utf-8 format)

Tables

Any data should work

Bug Reporting

Send any bugs or unanswered questions to the e-mail address included in your purchase confirmation or make an issue on the GitHub page. Please include your OS X version as well as your Sketch version in any bug reports.
